Project Wonderland Modules

Project Wonderland Modules are components which can be loaded dynamically to extend Project Wonderland capabilities. Example modules include 2D and 3D applications and non-visual components such as authentication plug-ins.

Announcements

A number of application modules are in development and will be released shortly. These modules include the following applications:

  • Video player application
  • VNC application
  • PDF Viewer application
  • Music in Wonderland 3D demo

About Project Wonderland Modules

Vision and Goals

Project Wonderland Modules provide a way to extend the Project Wonderland core with new run-time functionality. We're adopting a modular approach so that the community can create extensions to Wonderland that plug in using a documented module API. This API is currently being designed.

We'll release several Wonderland applications in this project using a preliminary version of the module system.
